# 信用卡积分规则

## 积分获取

### 常规消费
每消费1元人民币积1分。批发类、房产类、汽车类、医院类消费不积分。

### 多倍积分
- 生日当月消费享2倍积分
- 境外消费享3倍积分
- 指定商户（星巴克、麦当劳、中石油）享5倍积分

## 积分有效期
积分自获取之日起3年内有效，到期日为该年度12月31日。过期积分自动清零，不予补回。

## 积分兑换

| 兑换项目 | 所需积分 | 说明 |
|---------|---------|------|
| 年费减免 | 20000-80000 | 按卡种不同 |
| 航空里程 | 10000起 | 500里程起兑 |
| 话费充值 | 2500 | 50元面额 |
| 视频会员 | 5000 | 月卡 |

## 积分转赠
持卡人可将积分转赠给直系亲属（配偶、父母、子女），每年限转赠一次，单次上限100000分。需提供关系证明材料。
